Quick Tutorial

Putting a map in a web page is easy. Simply include the following in your page: 

      <iframe src="http://openlayers.org/viewer/"
        width="200px" height="200px"
        scrolling="no"
        marginwidth="0" marginheight="0"
        frameborder="0">
      </iframe>

You might want to adjust the width and height attributes, or you might want to put the whole thing inside of a div like this: 

    <div style="position:absolute; right:0px; top:150px; width:25%; height:200px;">
      <iframe src="http://openlayers.org/viewer/"
        width="100%" height="100%"
        scrolling="no"
        marginwidth="0" marginheight="0"
        frameborder="0">
      </iframe>
    </div>
